# ManageBac icon collection

Exact icons pulled from ManageBac's public asset CDN (`assets.managebac.com`),
mirrored here so we have local copies if we ever need to self-host or embed them.

The widget currently references the live CDN URLs directly (the widget CSP allows
`assets.managebac.com`), so these files are a reference copy, not load-bearing.
